# DragonBoard 410c rescue archives

These archives preserve the DragonBoard 410c rescue packages previously
published by 96Boards/Linaro. The original download endpoints are no longer
reliable, so build scripts consume the repository-owned copies through the
immutable `dragonboard410c-fw-v1` tag.

The archives contain their original license and checksum files. They are
redistributed unchanged; review the license inside each archive before reuse.
